## 📌 Projects

### 🌍 WanderLust — Full Stack Web Application (Major Project)

A complete vacation rental platform where users can **browse, book, and list properties** such as houses, villas, and farmhouses.

#### 🔑 Key Features
- Google & GitHub OAuth Authentication  
- Role-based Authorization (Users & Hosts)  
- Property Listing & Booking System  
- Advanced Search & Filters  
- User Reviews & Ratings  
- Real-time Location Mapping (Mapbox / Leaflet)  
- Image Upload with Cloudinary  
- Secure Sessions & Cookies  
- MVC Architecture  

#### 🧑‍💻 Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Bootstrap, EJS, React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas  
- **Authentication:** Passport.js, JWT  
- **Deployment:** Render
